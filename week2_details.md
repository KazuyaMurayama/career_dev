### アクション#25: RAGチャットボットデモアプリを作成 🤖 30分

**目的:** クライアント面談で実演するデモアプリ。「Claude Codeで30分でRAGが作れる」ことを証明する。

**Claude Codeプロンプト:**
```
Streamlitで社内FAQ RAGチャットボットのデモを作成して。

要件:
- Claude API (claude-sonnet-4-20250514) を使用
- PDFアップロード → テキスト抽出 → チャンク分割 → 簡易ベクトル検索
- ユーザーが質問を入力すると、アップロードしたPDFの内容を元に回答
- サイドバーにPDFアップロード機能
- メイン画面にチャットインターフェース
- サンプルPDF（社内FAQ風）も同梱

技術スタック:
- Python 3.11+
- Streamlit
- anthropic (Claude API)
- PyPDF2 (PDF読み込み)
- sentence-transformers (埋め込み) or シンプルなTF-IDF検索

ファイル構成:
- app.py (メインアプリ)
- requirements.txt
- sample_faq.pdf (サンプルデータ)
- README.md
```

### アクション#26: デモアプリをStreamlit Cloudにデプロイ 🤖 15分

**Claude Codeプロンプト:**
```
作成したRAGデモアプリをStreamlit Cloudにデプロイして。
- GitHubリポジトリにプッシュ
- Streamlit Cloudの設定ファイル（.streamlit/config.toml）を追加
- secrets.tomlにClaude API keyの設定方法をREADMEに記載
```

#### 26-A. GitHubリポジトリにプッシュ（Claude Codeで実行）
```bash
# Claude Codeに依頼:
cd rag-demo
git init
git add .
git commit -m "RAG chatbot demo"
git remote add origin https://github.com/[ユーザー名]/rag-demo.git
git push -u origin main
```

#### 26-B. Streamlit Cloudでデプロイ
1. [Streamlit Cloud](https://share.streamlit.io/) にアクセス
2. 右上の **「Sign in」** → **「Continue with GitHub」** をクリック → GitHubアカウントで認証
3. ダッシュボードが開いたら、右上の **「Create app」** をクリック
4. **「Deploy a public app from GitHub」** を選択
5. 以下を入力:
   - **Repository:** `[ユーザー名]/rag-demo` を選択
   - **Branch:** `main`
   - **Main file path:** `app.py`
6. **「Advanced settings」** をクリック → **「Secrets」** タブを開く
7. 以下をSecretsに入力:
   ```toml
   ANTHROPIC_API_KEY = "sk-ant-xxxxx（自分のClaude APIキー）"
   ```
8. **「Deploy!」** ボタンをクリック
9. 1〜3分待つとアプリが公開される → URLをコピー（例: `https://[ユーザー名]-rag-demo.streamlit.app`）
10. アプリが動作するか確認（PDFアップロード→質問→回答）

**完了条件:** URL取得・動作確認
