## 1. 自己PR（冒頭5分）

```
男座と申します。現在STANDARDでシニアマネージャーとして
生成AIサービスの立ち上げを担当しており、4月末に退職し、
6月から独立する予定です。

強みは「提言だけで終わらない」点です。
コンサルファームは戦略を作るが実装できない、
MLエンジニアは実装できるがビジネス提案が弱い、
という課題があります。私はAmazonで機械学習モデルにより
7億円の新規収益を創出し、STANDARDでは生成AIサービスを
6ヶ月で年商5,000万円に育てました。
戦略→実装→ROI測定を一人で完結できる点が差別化です。
```

## 2. 想定されるエージェントからの質問と回答例

**Q1. なぜ独立するのですか？**
```
大きく3つの理由があります。

1つ目は、より多くの企業のAI活用を支援したいからです。
1社専属では関われる課題に限界があります。
複数の企業に横断的に関わることで、
業界をまたいだ知見をクライアントに還元できます。

2つ目は、スピードです。
社内承認や組織の制約なしに動ける独立という形が、
AI活用の現場で最もインパクトを出しやすいと判断しました。
Claude Codeを使えば会議翌日にはプロトタイプをデモできます。
組織の意思決定スピードに縛られたくないという思いがあります。

3つ目は、実績の再現性を証明したいからです。
Amazonで7億円、STANDARDで5社ROI300%超という成果を、
自分のメソッドとして体系化し、より多くの企業に届けたいと考えています。
```

**Q2. 希望単価300万円の根拠は？**
```
3つの根拠があります。

【市場相場との比較】
同等のポジション（AI戦略×実装一気通貫）を
コンサルファームで調達すると月400〜800万円かかります。
私はその半額以下で、戦略提言と実装を同時にお届けします。
市場相場のAI事業化コンサルタントの上位レンジ（月250〜300万円）
とも合致した水準です。

【クライアントにとっての実質コスト】
稼働20〜50%でのご依頼が可能なため、
1社あたりの月額負担は60万円〜150万円です（単価300万円×稼働20〜50%）。
コンサルファームへの発注と比べ、
「シニア人材が直接動く」にもかかわらず大幅にコストを抑えられます。

【ROIの実績】
Amazon：機械学習モデルで新規収益7億円創出
STANDARD：5社でROI300%超（回収期間3年）を実現
これらの実績を踏まえると、月額300万円（フル稼働換算）は
クライアントにとって十分に合理的な投資です。
```

**Q3. 稼働率・稼働形態の希望は？**
```
基本は20〜60%、フルリモートを希望しています。
複数社と並行して関わることで、幅広い知見をご提供できます。
ただし案件内容次第では稼働率・形態は柔軟に相談できます。
```

**Q4. 得意な業界・案件タイプは？**
```
EC・デジタル広告（Amazon経験）、医療・製薬（M3経験）、
製造業・金融（STANDARD経験）が得意です。

案件タイプは「PoC止まりを本番化に持っていく」支援が最も得意です。

戦略・企画から実装、そして社内展開（研修・ワークショップを含む）まで
一気通貫で対応できる点が特徴です。

技術面では、生成AI（RAG・エージェント等）と
データサイエンス（分析・機械学習予測モデル等）含めた両方のDX企画実行へ対応できます。
```

**Q5. 稼働開始はいつから可能ですか？**
```
正式な稼働開始は2026年6月1日からです。
5月からの開始についてはご相談に応じます。
現職との兼ね合いを踏まえ、まずは週1〜2日から始め、
6月以降に稼働を本格化できる体制を整えます。
```

## 3. こちらからエージェントに確認すべきこと

```
いくつか確認させてください。

① AI/DSコンサルタントで月額200万円以上の案件は
  現在何件程度ありますか？また増えていますか？

② 稼働20〜50%・フルリモート限定にした場合、
  60万円〜150万円（単価300万円×稼働20〜50%）の案件数はどの程度に絞られますか？

③ 7月開始を前提に、今からエントリーや面談準備を
  進められる案件はありますか？

④ 単価交渉はどのタイミングで、誰が行いますか？
  エージェント経由ですか、直接ですか？

⑤ 御社が得意とする業界・企業規模を教えてください。
```

## 4. 要望として伝えるべき事項

```
いくつかご要望があります。

① 単価は300万円（フル稼働換算）を基準としてください。
  稼働率で調整しますが、単価自体は下げたくないです。

② 業務内容は「AI/DS戦略〜実装」に限定してください。
  単純なデータ加工・集計のみの案件はお断りしたいです。

③ 現職在籍中のため、6月末まではご紹介いただいた案件の
  面談を平日夜・土日でお願いします。

④ 複数エージェントに並行登録していることをご承知おきください。
```

## 競合との差別化ポイント（質問された場合）

| 比較対象 | 相手の弱点 | 男座さんの優位点 |
|---------|-----------|----------------|
| MBB等コンサルFA出身 | 戦略は作れるが実装できない、単価400〜600万円 | 実装まで一人で完結、300万円で同等以上の成果 |
| GAFA出身MLエンジニア | 技術力は高いがビジネス提案・経営報告が弱い | P&L責任・チームリード・経営層への説明実績あり |
| SIer・ベンダー出身 | 最新AI技術への対応が遅い、提案が自社製品依存 | 最新生成AI（RAG/Dify/Claude）実装経験、ベンダー中立 |
| コンサルファームへの依頼 | 月400〜600万円、担当者がジュニアになりがち | 月300万円、シニア人材が直接対応、意思決定が速い |
