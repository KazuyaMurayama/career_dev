## 日別スケジュール

### Day 1（水 4/2）— キックオフ・方針確認 ⏱️ 1時間

#### #1 LinkedInヘッドライン更新（30分）
**目的**: 検索でヒットするキーワードを盛り込み、第一印象を最大化する

**手順:**
1. LinkedIn（https://www.linkedin.com）にログイン
2. プロフィール写真下の鉛筆アイコン → 「基本情報を編集」
3. 「ヘッドライン」欄を以下に変更：

```
AI戦略コンサルタント｜元 Amazon シニアDS｜生成AI/ML→ROI 300%超の事業成果｜業務委託 受付中
```

4. 「保存」をクリック

**成果物**: ヘッドライン更新完了のスクリーンショット

---

#### #5 オープン・トゥ・ワーク設定（15分）
**目的**: エージェントや採用担当者から見つけてもらいやすくする

**手順:**
1. プロフィールページ → 「オープン・トゥ・ワーク」ボタン
2. 「フリーランス（業務委託）」を選択
3. 職種に「AIコンサルタント」「データサイエンティスト」「機械学習エンジニア」を追加
4. 表示範囲: 「採用担当者のみ」を選択（目立つ緑バッジを避けたい場合）または「全員」
5. 「保存」

**成果物**: 設定完了

### Day 4〜5（土日 4/5〜6）— エージェント3社登録 ⏱️ 4.5時間

#### #6 レバテックフリーランス登録（45分）
**目的**: 国内最大手・案件数最多のエージェントに登録し、面談枠を確保する

**URL**: https://freelance.levtech.jp/

**手順:**
1. 上記URLにアクセス → 「無料登録」
2. 基本情報を入力（氏名・メールアドレス・電話番号・現在の職種）
3. スキルシートをアップロード（Skill Sheet 男座員也 202601.docx）
4. 希望単価: **月額240万円〜300万円**（低く書かない）
5. 稼働可能時期: **2026年7月〜**（現職在籍中であることを明記）
6. 登録完了後、担当者から電話が来るので日程調整

**注意点:**
- 単価は「現在のスキルシート希望額300万円」ではなく、現実的に決まりやすい**240万円〜**で登録
- 「稼働率50%・フルリモート」を必ず明記

**成果物**: 登録完了メール受信確認
